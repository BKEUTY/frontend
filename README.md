<a name="english"></a>
## English

### Overview
BKEUTY is a modern beauty e-commerce platform designed to provide a seamless and aesthetic shopping experience. The application focuses on high-performance user interfaces, responsive design, and professional interactions using the Ant Design system.

### Key Features

#### 1. Authentication System
- **Login:** Secure access with email/password validation and social login options.
- **Registration:** Streamlined account creation with real-time field validation.
- **Forgot Password:** Complete recovery flow including email verification, OTP entry, and password reset.
- **Security:** Input validation for emails, passwords (strength check), and required fields.

#### 2. User Interface & Experience
- **Responsive Design:** Fully optimized for desktop, tablet, and mobile devices.
- **Split-Screen Layout:** Professional aesthetic for authentication pages with dynamic background integration.
- **Interactive Feedback:** Loading states, success/error notifications, and smooth transitions.
- **Brand Identity:** Consistent color palette (Deep Magenta) and typography (Be Vietnam Pro) across all screens.

#### 3. Core Functionality
- **Navigation:** Efficient routing system protecting authenticated routes and managing user flow.
- **Localization:** Bilingual support (English/Vietnamese) integrated throughout the application.

### Technical Stack
- **Frontend Framework:** React 19
- **UI Component Library:** Ant Design 6
- **State Management:** Context API
- **Routing:** React Router v7

## Tiếng Việt

### Tổng quan
BKEUTY là nền tảng thương mại điện tử chuyên về mỹ phẩm, được thiết kế để mang lại trải nghiệm mua sắm hiện đại và tinh tế. Ứng dụng tập trung vào giao diện người dùng hiệu suất cao, tương thích đa thiết bị và trải nghiệm chuyên nghiệp sử dụng hệ thống Ant Design.

### Tính năng chính

#### 1. Hệ thống Xác thực
- **Đăng nhập:** Truy cập an toàn với xác thực email/mật khẩu và tùy chọn đăng nhập mạng xã hội.
- **Đăng ký:** Tạo tài khoản nhanh chóng với kiểm tra dữ liệu đầu vào theo thời gian thực.
- **Quên mật khẩu:** Quy trình khôi phục đầy đủ bao gồm xác thực email, nhập mã OTP và đặt lại mật khẩu mới.
- **Bảo mật:** Kiểm tra chặt chẽ định dạng email, độ mạnh mật khẩu và các thông tin bắt buộc.

#### 2. Giao diện & Trải nghiệm Người dùng
- **Thiết kế Thích ứng (Responsive):** Tối ưu hóa hoàn toàn cho máy tính, máy tính bảng và điện thoại di động.
- **Bố cục Split-Screen:** Giao diện trang xác thực chuyên nghiệp kết hợp hình ảnh thương hiệu năng động.
- **Tương tác Phản hồi:** Trạng thái đang tải, thông báo thành công/lỗi và hiệu ứng chuyển cảnh mượt mà.
- **Nhận diện Thương hiệu:** Đồng bộ bảng màu (Hồng đậm) và phông chữ (Be Vietnam Pro) trên toàn hệ thống.

#### 3. Chức năng Cốt lõi
- **Điều hướng:** Hệ thống routing hiệu quả, quản lý luồng người dùng và bảo vệ các tuyến đường cần xác thực.
- **Đa ngôn ngữ:** Hỗ trợ song ngữ (Tiếng Anh/Tiếng Việt) đồng bộ trên toàn bộ ứng dụng.

### Nền tảng Kỹ thuật
- **Frontend Framework:** React 19
- **Thư viện Giao diện:** Ant Design 6
- **Quản lý Trạng thái:** Context API
- **Định tuyến:** React Router v7
